## Required softwares:

- Install Python 3.14 version.
- Install latest django framework.
- Create project and application django framework.
- Run manage.py runserver to run the localhost application.
- Install pycharm tool or visual studio code.

## Installation

Python requires [python.org](https://python.org/) v3.14 to download and install.

Install Python and check python version.

```
python --Version
```

Clone the project using git command...

```sh
git clone "https://github.com/rajkumar14/django_restframework.git"
cd django_restframework/tutorial
```

## Development

Want to contribute? Great!

Python + Web for fast developing.
Make a change in your file and instantaneously see your updates!

Open your favorite Terminal and run these commands.

First Tab makemigration create table:

```sh
manage.py makemigrations
manage.py makemigrations
manage.py createsuperuser
```

#### Running application localhost

Run Django project with appropriate location:

```sh
manage.py runserver
```




your preferred browser.

```sh
http://127.0.0.1:8000/
```
